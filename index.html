How to Play the Guessing Game
1. Objective: Guess the correct number between 1 and 10.
2. How to Play: Enter a number (1-10) in the input field.
Press the “Check” button to submit your guess.
3. Rules:
The game will randomly generate a number between 1 and 10.
If your guess matches the generated number, you'll see: “BOOM! You guessed the correct number!”
If your guess is incorrect, the game will tell you the correct number and encourage you to try again. For example: “Your guess (5) is not the right number; it's 7.”
If you enter a number outside the range 1-10, the game will warn you to: “Please input a valid number between 1 and 10.”
4. Pro Tip: Keep trying! You never know when you'll get lucky.
5. Have Fun!
